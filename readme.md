## Introduction

Halcyonic is a pixel-perfect **HTML & CSS UI Clone** of the popular **Halcyonic template by HTML5 UP**, focused on practicing real-world frontend layout, typography, spacing, and component styling using **pure HTML5 and CSS3 only (no JavaScript)**.

This project demonstrates a **real-world static website structure** with multiple pages such as **Homepage**, **Three Column**, **Two Column**, and **One Column**, all connected through clean navigation and reusable UI components.

---

## Screens Included

- **Homepage (Index)**: Hero intro with call-to-action, feature strips, and content area
- **Three Column Page**: Complex layout with a main content area flanked by two sidebars
- **Two Column Page**: Layouts featuring a sidebar on the left or right
- **One Column Page**: Full-width content page for text-heavy articles
- **Navigation Bar**: Top header navigation with logo
- **Footer Section**: Links, blurb area, and copyright info

---

## Features

### Homepage (Index)

- Acts as the main entry point of the website
- Full-width Hero Banner with "Call to Action" button
- "Features" strip with grid-based icons/images
- Content area with multi-column testimonials and info
- Fully responsive grid layout

---

### Three Column Page

- Complex grid layout demonstrating:
  - Left Sidebar (Navigation/Links)
  - Center Main Content
  - Right Sidebar (Additional Info)
- Clean typography and vertical rhythm
- Matches original Halcyonic layout style

---

### Two Column & One Column Pages

- **Two Column #1**: Wide left content area with a narrow right sidebar
- **Two Column #2**: Narrow left sidebar with a wide right content area
- **One Column**: Distraction-free full-width text layout
- Demonstrates versatility of the grid system

---

### Navigation

- Top-aligned navigation bar
- Clean link styling
- Hover effects matching the original design
- Consistent navigation across all pages

---

### Footer

- Multi-column link lists
- Informative text blurb section
- Copyright section with bottom alignment
- Pixel-matched spacing and background styling
